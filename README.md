# easy-mnist-kerasjs

This app predict the label of input mnist image whose size is 28x28.

You can try this app [here](https://easy-mnist-kerasjs.herokuapp.com/)!

## How to use

```
$ npm install
$ node index.js
```

You can access `http://localhost:8000` and upload your image file.
